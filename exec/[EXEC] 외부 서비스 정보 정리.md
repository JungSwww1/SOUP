## Liveblocks API

- Yjs 기반의 동시 편집 지원 라이브러리
- 마우스 커서 위치 공유, 마우스 커서 채팅, 페이지 내용 실시간 공유 등을 위한 Room Storage 활용
- 유저별 인증 후 특정 Room에 입장하기 위한 ID token 발급
